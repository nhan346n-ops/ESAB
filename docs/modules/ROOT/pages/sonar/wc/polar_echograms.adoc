= Polar Echograms

== Overview

The polar echogram is a tool allowing to create g3d file containing polar echograms of water column data. An echogram is a mean or max of values projected on a grid fitting a plane approximating the swath position.

The g3d file can be imported in globe and echograms can be displayed and selected in 2D and 3D views

== Algorithm parameters

* _Sample resolution_ : pixel size of the echogram image generated. Compute button retrieves a sample size based on the max sample interval.

* _Interpolation_ : true if linear interpolation should be applied to fill gaps in resulting image.

* _WC Filters_ can be set to discard specific echoes in the water column. It can be usefull to activate the specular filtering to remove all data around and below the main specular circle.
