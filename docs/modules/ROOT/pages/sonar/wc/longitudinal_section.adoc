= Longitudinal Slicer

== Overview
The longitudinal slicer is a tool allowing to create g3d file containing slices of water column data along the vessel track. A slice is a mean or max of values located around the location of the slice.

The g3d file can be imported in globe and slices can be displayed and selected in 2D and 3D views

== Algorithm parameters
* _Grid count_ and/or _Across spacing_
** _Grid count_ : expected number of slices. Spacing between them is computed with the min/max across ping distance for the file.
** _Across spacing_ (delta_across) : expected space between each slice. The number of slices is computed if grid_count = 0. If spacing is defined, slices are centered on navigation line. It is equal to the slice thickness.

* _Sample size_
** _Vertical sample size_ (delta_elevation) : vertical distance between each line of the slice image generated. Compute button retrieves a vertical sample size based on the max sample interval.

** _Alongtrack sample size_ (delta_along) : alongtrack distance between each column of the slice image generated. Compute button retrieves a horizontal resolution based on spatial dispersion of detections.

* _Interpolation_ : true if linear interpolation should be applied to fill gaps in resulting image.

* _WC Filters_ can be set to discard specific echoes in the water column. It can be usefull to activate the specular filtering to remove all data around and below the main specular circle.
