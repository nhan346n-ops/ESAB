= Orpheus

== Overview

Orpheus is able to generate plots where one can visualize sounder data files statistics (.xsf.nc) compared to a reference file (.dtm.nc) and export results as csv file.

Orpheus can compute and plot the total vertical uncertainties for some selected orders (See https://iho.int/uploads/user/pubs/standards/s-44/S-44_Edition_6.1.0.pdf[International Hydrographic Organization - Standards for Hydrographic Surveys]). This orders are :
[options="header"]
[cols="20h,~"]
|=======================
|Order |Description
|Order 2 |Areas where a general description of the sea floor is considered adequate
|Order 1b |Areas where underkeel clearance is not considered to be an issue for the type of surface shipping expected to transit the area
|Order 1a|Areas where underkeel clearance is considered not to be critical but features of concern to surface shipping may exist
|Special Order|Areas where underkeel clearance is critical
|Exclusive Order|Areas where there is strict minimum underkeel clearance and manoeuvrability criteria
|=======================
