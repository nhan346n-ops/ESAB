= Recompute Detection Position (Raytracing)

🚧 Documentation in progress 🚧

== Overview

This process allows to recompute the 3D position of bathymetry detections (x,y,z) from the recorded two-way travel times. +
It involves *beam vector* computation, and *raytracing* through the water column.

== Algorithms

[cols="1,3,1",options="header"]
|===
| Algorithm | Description | Reference
| *VCCA* + 
Virtual Concentric Cone Algorithm
a| [none] 
* Tx and Rx arrays : co-located
* beam pattern : oriented triangles
* ✅ static angular separation (installation alignment angles)
* ✅ dynamic angular separation (beam steering angles, platform yaw, pitch and roll)
* ❌ static position separation (installation offsets)
* ❌ dynamic position separation (platform displacement)
| https://journals.lib.unb.ca/index.php/ihr/article/download/20675/23837/[Beaudoin, J. D., Clarke, J. H., & Bartlett, J. E. (2004). Application of surface sound speed measurements in post-processing for multi-sector multibeam echosounders. The International hydrographic review].
| *NCCA* + 
Non-Concentric Cone Algorithm
a| [none] 
* Tx and Rx arrays : separeted (staticly and dynamicly)
* beam pattern : oriented cones
* ✅ static angular separation (installation alignment angles)
* ✅ dynamic angular separation (beam steering angles, platform yaw, pitch and roll)
* ✅ static position separation (installation offsets)
* ✅ dynamic position separation (platform displacement)
| Hamilton, T., Beaudoin, J., & Clarke, J. H. (2014, April). A more precise algorithm to account for non-concentric multibeam array geometry. In Canadian Hydrographic Conference.
| *NCTA* +
Non-Concentric Triangle Algorithm
a| [none] 
* Tx and Rx arrays : separeted (staticly and dynamicly)
* beam pattern : oriented triangles
* ✅ static angular separation (installation alignment angles)
* ✅ dynamic angular separation (beam steering angles, platform yaw, pitch and roll)
* ✅ static position separation (installation offsets)
* ✅ dynamic position separation (platform displacement)
| https://www.sciencedirect.com/science/article/am/pii/S0029801819309539[Bu, X., Yang, F., Ma, Y., Wu, D., Zhang, K., & Xu, F. (2020). Simplified calibration method for multibeam footprint displacements due to non-concentric arrays. Ocean Engineering, 197, 1068620.]
|===
