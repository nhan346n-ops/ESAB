== Xsf GNSS tide

=== Overview
This processus computes an estimated tide based on sensor data from XSF files. Sensor data (gnss info extracted from NMEA fields or directly from height sensors) are first converted relatively to a reference surface (LAT or ZeroHydro) then a lowpass filter is applied to generate the estimated tide.
