= CTD HOV (.cnv) -> Sensor-netCDF

== Overview
This converter is designed to convert CTD HOV sensor data from CSV files (.cnv) into the NetCDF format.

== NetCDF File Structure
**Dimensions**

- `time`: Timestamp index

**Coordinates**

- `time`: Nanoseconds since 1970-01-01 (calendar: gregorian)
- `latitude` (if navigation provided) : Decimal degrees north `[-90, 90]`
- `longitude`(if navigation provided) : Decimal degrees east `[-180, 180]`

**Data Variables**

- `depth` — Depth below sea surface [m]
- `temperature` — Water temperature [°C]
- `salinity` — Practical salinity [PSU]
- `turbidity` — Water turbidity [FTU]
- `pressure` — Pressure [dbar]

**Metadata Variables**

 - `crs` — Coordinate reference system
