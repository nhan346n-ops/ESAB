= CSV -> Sensor-netCDF

== Overview
This converter is designed to convert sensor data from CSV files into the NetCDF format.

== NetCDF File Structure

The content of the netCDF file will depend on the columns selected in the input CSV file.

**Dimensions**

- `time`: Timestamp index

**Coordinates**

- `time`: Nanoseconds since 1970-01-01 (calendar: gregorian)
- `latitude`: Decimal degrees north `[-90, 90]`
- `longitude`: Decimal degrees east `[-180, 180]`

**Data Variables**

- `depth` — Depth below sea surface [m]
- `temperature` — Water temperature [°C]
- `conductivity` — Electrical conductivity [S/m]
- `salinity` — Practical salinity [PSU]
- `sound_velocity` — Speed of sound [m/s]
- `turbidity` — Water turbidity [FTU]
- `pressure` — Pressure [dbar]

**Metadata Variables**

 - `crs` — Coordinate reference system
