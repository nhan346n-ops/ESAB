= CTD HOV (.asc) -> Sensor-netCDF

== Overview
This converter is designed to convert CTD sensor data from CSV files (.asc) into the NetCDF format.

== NetCDF File Structure
**Dimensions**

- `time`: Timestamp index

**Coordinates**

- `time`: Nanoseconds since 1970-01-01 (calendar: gregorian)
- `latitude` (from shapefile) : Decimal degrees north `[-90, 90]`
- `longitude` (from shapefile) : Decimal degrees east `[-180, 180]`

**Data Variables**

- `depth` ‚Äî Depth below sea surface [m]
- `temperature` ‚Äî Water temperature [¬∞C]
- `conductivity` ‚Äî Electrical conductivity [S/m]
- `salinity` ‚Äî Practical salinity [PSU]
- `sound_velocity` ‚Äî Speed of sound [m/s]
- `turbidity` ‚Äî Water turbidity [FTU]
- `pressure` ‚Äî Pressure [dbar]

**Metadata Variables**

 - `crs` ‚Äî Coordinate reference system
