= Cross Compare

== Overview

This tool identifies intersection points and compares data around them.

The process may be done on :

* single list of files : the navigation route is computed from all the input files. Intersections are those of this navigation route with itself.
* two lists of files : two different navigation routes are computed, one on each list of input files. Intersections are those which occurs between those two routes.
