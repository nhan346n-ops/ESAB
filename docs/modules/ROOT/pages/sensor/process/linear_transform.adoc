= Linear transformation

== Overview

This application takes Sensor-netCDF file(s) and process the layers by a linear transformation ( ``y = a * x + b ``).

By default, the path of the output file will be i_path with the suffix "-linear_transform".
The factor `a`, and `b` will be respectively `1` and `0`.
