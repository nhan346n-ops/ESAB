== Longitudinal slicer

=== Overview
The horizontal slicer is a tool allowing allowing to create g3d file containing slices of sensor netCDF. A slice is a mean or a max of values located around the location of the slice.

The g3d file can be imported in globe and slices can be displayed and selected in 2D and 3D views
