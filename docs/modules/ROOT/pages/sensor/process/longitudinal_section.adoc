== Longitudinal slicer

=== Overview
The longitudinal slicer is a tool allowing to create g3d file containing slices of sensor netCDF data along a path. A slice is a mean or max of values located around the location of the slice.

The g3d file can be imported in globe and slices can be displayed and selected in 2D and 3D views
