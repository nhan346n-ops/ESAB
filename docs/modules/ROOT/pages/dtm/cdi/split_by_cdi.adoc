= Split by CDI

== Overview

This application takes DTM file(s) in new format as input and returns as much
CDIs as there are in the input file(s). The program creates one file for one CDI named with the CDI as suffix.
If there is cells with elevation but without CDI, an extra file is created with "-missing" reseting all cells with CDI.
For the cdi_index layer, set it to missing_value or 0 in function of the selected CDI.
For the cdi_reference layer, put the selected CDI on the "0" index and remove the others.
