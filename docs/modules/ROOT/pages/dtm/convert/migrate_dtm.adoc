= Legacy DTM -> DTM netCDF4

== Overview

This application takes DTM file(s) as an input and will migrate it (or them) into a new netcdf file format.

[WARNING]
The output file name is the same as the input file name with a ".nc" added at the end its name.
No test is done for previously existing file names
