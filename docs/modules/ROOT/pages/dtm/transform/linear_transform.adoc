= Linear Transformation

== Overview

This application takes DTM file(s) and process the elevation, elevation_min, elevation_max and stdev layers by a linear
transformation ( y = a * x + b ) in new DTM file(s).

By default, the path of the output file will be i_path with the suffix "-linear_transform". The factor a, and b will be respectively
1 and 0.
