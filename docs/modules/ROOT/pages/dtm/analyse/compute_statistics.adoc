= Statistics

== Overview

This process computes statistics on the different layers of one or more DTMs.

== Input

One or more DTM files (.dtm.nc).

== Parameters

**Scope** : statistics can be computed by file or globally.

**Computed statistics** :

* Mean
* Standard deviation
* Median
* Data distribution (histograms)
* Confidence levels :
** from a **percentage** : computes an interval that contains X% of the data. This interval excludes extreme data, so it centered on the median. For example, if the percentage is 95%, the result interval excludes the 2.5% lower and higher data.
** from **min / max values** : calculates the percentage of data within the interval defined by these values.

== Output
Statistics are printed in console, and can be exported in a new CSV file.
The histograms can be displayed in dialog windows, and also exported as image files.
