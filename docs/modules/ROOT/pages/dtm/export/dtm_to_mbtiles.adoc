= DTM -> MBtiles

== Overview

This application takes **DTM** file(s) as input and will export it (or them) as **MBtiles** file(s), using  https://gdal.org/en/stable/drivers/raster/mbtiles.html[GDAL MBtiles driver] driver capabilities .

Mainly used for exporting bathymetry to https://opencpn.org/wiki/dokuwiki/doku.php?id=opencpn:manual_advanced:charts:mbtiles[**OpenCPN**].
